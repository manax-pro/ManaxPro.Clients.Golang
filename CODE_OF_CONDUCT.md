# Code of Conduct

This project is intended to be a safe and productive place for contributors and users.

The following principles apply to all interactions in issues, pull requests, discussions and any other project spaces.

## Expected Behavior

- Be respectful and professional in all communications.
- Assume good intent, but do not ignore harm.
- Welcome questions and different viewpoints.
- Offer constructive feedback and focus on the technical topic.
- Respect time and boundaries of other contributors.

## Unacceptable Behavior

Examples of unacceptable behavior include, but are not limited to:

- Harassment, discrimination or personal attacks.
- Threats or incitement of violence.
- Derogatory comments related to personal characteristics.
- Sustained disruption of discussions.
- Sharing private information about others without explicit permission.

## Scope

This Code of Conduct applies within all project spaces, including:

- Issue trackers and pull requests.
- Discussions, forums and chat channels.
- Any public interaction where you represent this project.

## Reporting

If you experience or witness unacceptable behavior, you can report it by opening a confidential issue or by contacting the maintainers privately (for example via the email listed in the repository).

Provide as much context as you can, including:

- What happened.
- Where it happened (issue number, link, etc.).
- When it happened.
- Any other information that might help us understand the situation.

## Enforcement

- Maintainers will review all reports as quickly as reasonably possible.
- Actions may include a private warning, temporary restrictions, or permanent removal from project spaces.
- Enforcement decisions are made to protect the community and keep the project healthy.
